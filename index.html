Usability

This portfolio is created with many features to make the user experience more convinient. The image and video in this site have alternative text in case they do not load. The color of links have been changed so that they adequetely contrast with the gray background, helping visually impared users. Links keep the user on the same tab, allowing them to hit the back button and return to the website. The text on the page is centered allowing for a more structured view on mobile and no desktop. There are also breaks between between images and text so the whole websites is easier to scroll through.
